여러 개의 문자열로 구성되 배열이 주어졌을 때 각각의 문장을 먼저 정렬하고 그 다음에 전체 문자열을 사전순으로 다시 정렬하는 알고리즘이 있다고 가정. 이 알고리즘의 수행시간은?

1. 가장 긴 문자열의 길이는 s
2. 배열의 길이는 a

각 문자열을 정렬하는데 걸리느 시간 O(slogs).
모든 문자열 정렬하는 데 걸리는 시간 O(a*slogs).

이제 전체 문자열을 사전순으로 다시 정렬해야 함.

전체 배열의 문자열을 정렬하는데는 각각의 문자열을 비교할 때 걸리 수 있느 시간 O(s)와 배열을 정렬하느 시간 O(aloga)가 걸리므로 O(s*aloga)가 됨.

총 시간 복잡도는 O(a*s(loga + logs)).
